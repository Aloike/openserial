    Welcome to the OpenSerial project's main page !

    This project aims to provide a cross-platform, customizable, free and open-source Serial communication program.

    Some of its main advantages are :

    • A Plugin-based mechanism providing the ability to extend the software's capabilities, or use it as a base to develop your own code.
    • A cross-platform interface, which means you'll find the same software on Linux and Windows.
    • Usage of XML configuration files to save the settings, so you can save your configuration and share it between computers.
    • Command-line execution taking advantage of arguments to specify some useful parameters of the application.
    • And of course, it's FREE !
    This project is based on one of my "sandbox" softwares, "Sercomm", which was hosted on Sourceforge. My aim now is to produce a more structured project to improve its maintenability, reliability and quality, with hope it'll be helpful to someone.
